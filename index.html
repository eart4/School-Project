Что же такое FrontEnd разработка?

Существует всего два направления в программировании. Это FrontEnd и BackEnd. Сегодня я вам расскажу о FrontEnd разработке. FrontEnd-разработка — это создание пользовательского интерфейса на клиентской стороне веб‑сайта или приложения. Это всё, что видит пользователь, когда открывает веб-страницу, и с чем он взаимодействует: кнопки, баннеры и анимация. В процессе работы фронтендер взаимодействует с дизайнерами, бэкенд-разработчиками и UX-аналитиками. Чтобы понять, что собой представляет фронтенд, откройте страницу любого сайта — вы увидите интерфейс. Щёлкните правой кнопкой мыши по странице, выберите пункт «Исходный код страницы», и перед вами откроется её исходный код.

Редактор кода для FrontEnd разработки.

Существует много различных инструментов для написания кода FrontEnD разработки. Например: IntelliJ IDEA,VSCode,Sublime Text, Atom и т.п. Но сегодня я расскажу вам только об одном, а именно о VISUAL STUDIO CODE. Это один из самых популярных и простых редакторов кода для разработки полноценного сайта, созданный компанией Microsoft для MacOs, Windows и Linux. Он создан для разработки веб и облачных приложений. Включает в себя отладчик, инструменты для работы с Git, подсветку синтаксиса, IntelliSense и средства для рефакторинга. Имеет широкие возможности для кастомизации.

Какие же языки нужны для создания собственного сайта?

HTML

HTML язык гипертекстовой разметки — стандартизированный язык гипертекстовой разметки документов для просмотра веб-страниц в браузере. Веб-браузеры получают HTML документ от сервера по протоколам HTTP/HTTPS или открывают с локального диска, далее интерпретируют код в интерфейс, который будет отображаться на экране монитора. HTML (HyperText Markup Language) говорит браузеру, каково содержание страницы, например, «заголовок», «параграф», «список», «элемент списка».

CSS

CSS каскадные таблицы стилей — формальный язык описания внешнего вида документа (веб-страницы), написанного с использованием языка разметки (чаще всего HTML или XHTML). Также может применяться к любым XML-документам, например, к SVG или XUL. CSS (Cascading Style Sheets) говорит браузеру, как отображать элементы, например, «после первого параграфа отступ в 20 пикселей» или «весь текст в элементе body должен быть темно-серым и написан шрифтом Verdana».

JavaScript

JavaScript — мультипарадигменный язык программирования. Поддерживает объектно-ориентированный, императивный и функциональный стили. Является реализацией спецификации ECMAScript (стандарт ECMA-262). JavaScript говорит браузеру, как реагировать на некоторые взаимодействия, используя легкий язык программирования.

Оптимизация сайта

После написания свое веб-сайта, его нужно будет оптимизировать. Я отвечу на три вопроса касающийся оптимизации сайта. Что такое оптимизация? Для чего нужна оптимизация? Как оптимизировать сайт?

Оптимизация - это устранение недочетов и разработки более эффективного алгоритма работы. В результате которого снижаются издержки и повышается качество продукции.

Оптимизация сайта нужна для того чтобы человек смог быстро загружаться на сайт. Например что бы сайт даже при плохом интернете мог спокойно загрузится. Скорость сайта является ключевым фактором, определяющим его популярность. Рекомендуемое Google время загрузки страницы составляет менее двух секунд. Почему? Около 50% посещений мобильных сайтов прекращаются, если страницы загружаются дольше 3 секунд. Есть несколько способов оптимизации загрузки сайта.

1-й способ заключается в упрощения кода CSS или JavaScript. Много кода - не значит что это правильный код. Чем меньше кода, тем меньше весит ваш сайт, соответственно он будет быстрее загружаться.

2-й способ это оптимизация картинок, но как же это сделать? Существует множество сайтов которые позволяют их оптимизировать, то есть они их сжимают в качестве и следовательно они меньше весят.
